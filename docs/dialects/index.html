方言¶

** dialect **は、SQLAlchemyがさまざまなタイプの：term： `DBAPI`実装とデータベースと通信するために使用するシステムです。以下のセクションでは、各バックエンドの使用方法やさまざまなDBAPIのメモに固有の参照ドキュメントと注意事項が記載されています。

すべての方言では、適切なDBAPIドライバがインストールされている必要があります。

外部の方言¶

バージョン 0.8 で変更: SQLAlchemy 0.8以降、いくつかの方言が外部プロジェクトに移され、新しいデータベース用の方言も外部プロジェクトとして公開されます。ここでの根拠は、基本的なSQLAlchemyのインストールとテストスイートが過度に大きくならないようにすることです。

SQLite、MySQL、PostgreSQL、Oracle、SQL Server、Firebirdなどの古典的な方言は、当面はコアに残ります。

バージョン 1.0 で変更: Drizzle方言は第三者システムに移動されました。

SQLAlchemyの現在の外部ダイアレクトプロジェクトは次のとおりです。

プロダクションレディ¶

  • `ibm_db_sa <http://code.google.com/p/ibm-db/wiki/README> `_ - IBM DB2とInformix用のドライバ.IBMとSQLAlchemyの開発者が共同開発しました。
  • `sqlalchemy-redshift <https://pypi.python.org/pypi/sqlalchemy-redshift> `_ - Redshiftの_ _ドライバは、既存のPostgreSQL / psycopg2ドライバを採用しています。
  • `sqlalchemy-hana <https://github.com/SAP/sqlalchemy-hana> `_ - SAP Hanaのドライバ。
  • `sqlalchemy_exasol <https://github.com/blue-yonder/sqlalchemy_exasol> `_ - EXASolutionのドライバ。
  • `sqlalchemy-sqlany <https://github.com/sqlanywhere/sqlalchemy-sqlany> `_ - SAP用のSybaseドライバSQL Anywhere。SAPによって開発されました。
  • `sqlalchemy-monetdb <https://github.com/gijzelaerr/sqlalchemy-monetdb> `_ MonetDBのためのドライバ。
  • `snowflake-sqlalchemy <https://github.com/snowflakedb/snowflake-sqlalchemy> `_ Snowflakeのドライバ<https://www.snowflake.net/> `_。
  • `sqlalchemy-tds <https://github.com/m32/sqlalchemy-tds> `_ pythone-tdsの上にMS-SQL用の_ドライバ<https://github.com/denisenkom/pytds> `_。
  • `クレート<https://github.com/crate/crate-python> `_ CrateDB用ドライバ<https://crate.io/> `_。

実験的/不完全な¶

不完全な状態にあるか、やや実験的であると考えられる方言。

  • `カルチパン<https://bitbucket.org/zzzeek/calchipan/> `_ - Pandasを適応させる<http://pandas.pydata.org/> `_ dataframes to SQLAlchemy。
  • `sqlalchemy-cubrid <https://bitbucket.org/zzzeek/sqlalchemy-cubrid> `_ - CUBRIDデータベースのドライバ。

屋根裏¶

&quot;屋根裏部屋&quot;の方言は、ずっと前にSQLAlchemyに寄稿されたものですが、それ以来ほとんど注目を集めていませんでした。そして今半分の状態で自分のリポジトリに移動しています。これらの方言に興味のあるコミュニティメンバーは、現在のコードベースを自由に取り上げて作業ライブラリに自由に置くべきです。

  • `sqlalchemy-access <https://bitbucket.org/zzzeek/sqlalchemy-access> `_ - Microsoft Access用のドライバ。
  • sqlalchemy-drizzle <https://bitbucket.org/zzzeek/sqlalchemy-drizzle> `_ Drizzle MySQLの亜種のための _ - ドライバ。
  • `sqlalchemy-informixdb <https://bitbucket.org/zzzeek/sqlalchemy-informixdb> `_ informixdb DBAPIのドライバ。
  • `sqlalchemy-maxdb <https://bitbucket.org/zzzeek/sqlalchemy-maxdb> `_ MaxDBデータベースのドライバ
