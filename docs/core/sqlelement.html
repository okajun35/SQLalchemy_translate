`` CASE``式を生成します。

    WHEN (name = :name_1) THEN :param_1
    WHEN (name = :name_2) THEN :param_2

END

            where(
                case(
                    {"wendy": "W", "jack": "J"},
                    value=users_table.c.name,
                    else_='E'
                )
            )
